## Collaborator Note

- `docs/all-training/` contains authored source material for events, bundles, learning paths, self-study content, and external resources.
- `docs/explore/`, `docs/events/upcoming.md`, `docs/events/past-events.md`, `docs/external-resources.md`, and the training-catalogue alias pages are generated from that source material by the scripts in `scripts/generators/`.
- Edit source files under `docs/all-training/` when changing content structure or metadata; avoid hand-editing generated pages unless the page is explicitly maintained by hand.
- Rebuild generated pages with `.venv/bin/python scripts/generate_all.py`.
