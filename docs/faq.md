# Frequently Asked Questions

Common questions and answers about our training.

Common questions about HPC training and resources.

## Getting Started

??? question "How do I get access to HPC resources?"

    Access to NAISS HPC resources is typically granted through:

    - Research projects with computational needs
    - Educational programs and courses
    - Collaboration agreements

    Contact your local NAISS representative or visit [NAISS website](https://www.naiss.se/) for application procedures.

??? question "What if I've never used Linux before?"

    Don't worry! We have resources for beginners:

    - [Linux basics tutorial](external-resources.md)
    - [Beginner learning path](explore/learning-paths/beginner.md)
    - Introductory workshops (check [upcoming events](../events/upcoming.md))

??? question "Do I need programming experience?"

    Basic programming knowledge helps, but isn't always required. We offer training for various skill levels:

    - **Beginners**: Start with our [beginner path](explore/learning-paths/beginner.md)
    - **Intermediate**: Build on existing programming skills
    - **Advanced**: Focus on optimization and parallel programming

## Training and Events

??? question "How do I register for workshops?"

    Registration is typically handled through:

    - NAISS training portal
    - Event-specific registration forms
    - Email invitations for targeted audiences

    Check the [upcoming events](../events/upcoming.md) page for current registration links.

??? question "Are the workshops free?"

    Most NAISS training events are free for researchers and students at Swedish universities and research institutions. Some advanced or specialized training may have fees.

??? question "Can I get a certificate?"

    Certificates of completion are provided for many of our courses and workshops. These can be useful for your CV or institutional requirements.

## Technical Questions

??? question "What software is available on the clusters?"

    Available software varies by cluster, but typically includes:

    - Compilers (GCC, Intel, NVIDIA HPC SDK)
    - MPI implementations (OpenMPI, Intel MPI)
    - Mathematical libraries (BLAS, LAPACK, FFTW)
    - Python/R environments with scientific packages
    - Visualization tools

    Check the specific cluster documentation for available modules.

??? question "How do I run parallel jobs?"

    We support several parallel programming models:

    - **OpenMP**: Shared memory parallelism
    - **MPI**: Distributed memory parallelism
    - **Hybrid**: Combining OpenMP and MPI

    See our [parallel programming courses](explore/tags/parallel-performance.md) for details.
