# Parallel Python with MPI

## Summary

Learn how to scale Python applications across multiple nodes using MPI and `mpi4py`. This workshop introduces core parallel programming concepts and shows how to run distributed jobs efficiently on HPC clusters.

## Learning objectives

By the end of this workshop, you will be able to:

- Run parallel Python jobs using Slurm.
- Write basic MPI programs with `mpi4py`.
- Distribute workloads across multiple processes.
- Debug and troubleshoot common MPI issues.
- Understand scaling and performance trade-offs.

## Prerequisites

- Basic Python programming skills.
- Familiarity with the Linux command line.
- Access to an HPC cluster.
- Prior completion of Intro to HPC or Slurm Basics is recommended.
