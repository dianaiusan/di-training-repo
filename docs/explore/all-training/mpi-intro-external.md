# An introduction to parallel programming using Message Passing with MPI

Message Passing is presently a widely deployed programming model in massively parallel high performance computing. The concepts behind message passing and distributed memory computing will be introduced and the syntax of the key MPI calls will be explained.

## Course Details

**Dates**: 20, 21, 27 and 28 May 2026  
**Location**: Online  
**Duration**: 4 days  
**Level**: Beginner to Intermediate

## Summary

This course introduces the concepts behind message passing and distributed memory computing. You'll learn the syntax of key MPI calls and how to write parallel programs for HPC systems.

## Learning objectives

- Understand distributed memory parallelism
- Master MPI communication patterns
- Write and debug MPI programs
- Optimize MPI applications
- Use MPI in real-world applications

## Topics Covered

### Days 1-2: MPI Fundamentals
- Parallel computing concepts
- MPI initialization and finalization
- Point-to-point communication
- Blocking vs non-blocking operations

### Days 3-4: Advanced MPI
- Collective communications
- MPI data types
- Performance optimization
- Debugging parallel programs

## Prerequisites

- Programming experience in C, C++, or Fortran
- Basic Linux command line knowledge
- Understanding of parallel computing concepts (helpful)

## Target Audience

- Researchers developing parallel applications
- Students learning high-performance computing
- Developers optimizing scientific code
