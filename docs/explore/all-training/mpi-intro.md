# An introduction to parallel programming using Message Passing with MPI

## Summary

Message Passing is a widely deployed programming model in massively parallel high performance computing. It is suitable for a wide range of computer architectures, from multi-core desktop systems to the fastest HPC systems with several hundred thousand processing elements.

This online course is at beginner level and assumes no prior experience in parallel computing. The concepts behind message passing and distributed-memory computing are introduced, and the syntax of key MPI calls is explained. The course includes point-to-point communication, non-blocking communication, and collective communication calls.

Practical sessions are included to deepen understanding of the lectures. At the end of the course, participants should be able to write their own MPI programs at an intermediate level. The teaching language is English.

## Learning objectives

- Understand the concepts behind message passing and distributed-memory computing
- Use key MPI calls in practical examples
- Implement point-to-point and non-blocking communication patterns
- Use collective MPI communication effectively
- Write and run MPI programs at an intermediate level
