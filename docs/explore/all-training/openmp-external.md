# Introduction to Parallel Programming with OpenMP (External Resource)

This external course from the OpenMP Architecture Review Board provides comprehensive training in shared memory parallel programming.

## Course Description

Learn to harness the power of shared memory parallelism using OpenMP, the industry-standard API for parallel programming on multicore systems.

## Topics Covered

- OpenMP fundamentals and execution model
- Work sharing constructs (parallel, for, sections)
- Data sharing attributes and synchronization
- Performance optimization techniques
- Integration with existing code

## Why This External Course?

This official OpenMP training provides vendor-neutral, standards-based instruction that complements our hands-on HPC-focused courses.
