# Introduction to Python and Using Python in an HPC environment

The course is given under [NAISS](https://www.naiss.se/), by application experts from [UPPMAX](https://www.uu.se/centrum/uppmax/), [HPC2N](https://www.hpc2n.umu.se), [LUNARC](https://www.lunarc.lu.se/), [C3SE](https://www.c3se.chalmers.se/), and [InfraVis](https://infravis.se/).

## Summary

This online 4-day workshop aims to give a brief, but comprehensive introduction to using Python in a Swedish academic High-Performance Computing (HPC) environment. It covers a wide range of levels in using Python and you can decide which days to follow.

If you are a complete beginner, perhaps the first day(s) is sufficient for you, but you may follow the other days as well for inspiration. If you already are a Python programmer you could still benefit from attending the login session the first day.

## Learning objectives

- Log in to HPC clusters and work with Python modules.
- Use core Python concepts: variables, expressions, and statements.
- Use site-installed packages and install packages with pip/conda.
- Work with virtual environments.
- Request compute resources and write Python batch scripts.
- Use IDEs and Jupyter for scientific workflows.
- Perform basic analysis with Matplotlib, Pandas, and Seaborn.
- Understand HPC-specific Python topics: parallelism, GPUs, big data, and ML/DL.

## Prerequisites

- Familiarity with the LINUX command line.
- Basic Python is recommended but not required.

## Day-by-day content

### Day 1

- Log in to a cluster.
- Load the Python module.
- Use variables, expressions, and statements in Python.

### Day 2

- Load modules for site-installed Python packages.
- Use and install Python packages (pip and/or conda).
- Use virtual environments.
- Request compute resources and write batch scripts for Python.
- Learn about IDEs.

### Day 3-4

- Do basic analysis with Matplotlib in Jupyter.
- Learn more about compute resources, including parallelism and GPUs.
- Use Pandas and Seaborn.
- Work with big data and Python.
- Use Python for ML/DL.
- Guest lecture by Anders Hast and InfraVis introduction.

## Participation and compute resources

Participation is free and open to all users within NAISS, whether their site has NAISS resources or not.

Please make sure you have an account at SUPR if you want to participate in hands-on activities. A course project can be used to run examples if you do not already have HPC access.

Tetralith at NSC is mainly used for demos, but written material is intended to support users on resources they already use across Sweden.

### Course project policies

- You may receive access before the course. Do not run personal production jobs in the course project before training starts.
- The project remains open for 1-2 weeks after the course for course-related follow-up runs.
- Misuse of the allocation for non-course production runs may result in removal from the course project.
- Temporary storage may be provided. Store only course-related data and copy anything you need to keep soon after the course, since cleanup happens about a month later.

The course uses computing resources provided by the National Academic Infrastructure for Supercomputing in Sweden (NAISS) at NSC, partially funded by the Swedish Research Council through grant agreement no. 2022-06725.

If affiliated with UmU, SLU, LTU, MIUN, or IRF, you can use HPC2N systems instead of NSC. If affiliated with LU, you can use LUNARC resources. If affiliated with UU, you can use UPPMAX resources.
