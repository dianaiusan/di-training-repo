# Scaling Experiments with SLURM Job Arrays

## Summary

SLURM job arrays allow you to run many similar jobs — such as parameter sweeps or replicated simulations — without submitting each one individually. This course covers how to use job arrays efficiently, avoid common pitfalls like scheduler overload, and structure batch experiments cleanly.

## Learning objectives

- Understand when and why to use SLURM job arrays
- Submit, monitor, and cancel job arrays
- Use array indices to parameterise experiments
- Structure input/output for array jobs
- Avoid common mistakes that stress the scheduler

## Prerequisites

- Basic SLURM job submission (naiss-intro or equivalent)
