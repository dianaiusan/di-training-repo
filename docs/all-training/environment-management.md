# Software Environment Management on HPC

## Summary

Managing software dependencies is one of the most common sources of frustration on HPC systems. This course covers the LMOD module system used on NAISS clusters, Conda environments for Python and R, and Python virtual environments — and shows how to use these tools together to create reproducible, portable setups for your research code.

## Learning objectives

- Load, search, and manage software with LMOD modules
- Create and manage Conda environments for Python and R
- Use Python virtual environments (venv/pip)
- Export and reproduce environments across systems
- Understand when to use modules vs Conda vs containers

## Prerequisites

- Basic Linux command line and HPC familiarity (naiss-intro or equivalent)
