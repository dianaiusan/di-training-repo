# Bianca In-Depth: Improve Your Handling of Sensitive Research Data

Terminal login, file transfer from a terminal, advanced Slurm, and installing custom software and packages.

## Course Details

**Date**: 22 May 2026  
**Location**: Online  
**Duration**: 1 day  
**Level**: Intermediate

## Summary

This course focuses on effectively using the Bianca system for sensitive research data. You'll learn advanced techniques for secure data handling, terminal operations, and software management on sensitive data systems.

## Learning objectives

- Master terminal login to Bianca
- Learn secure file transfer techniques
- Understand advanced SLURM usage for sensitive data
- Install and manage custom software packages
- Follow best practices for sensitive data handling

## Topics Covered

- Bianca system architecture and security
- Terminal-based login procedures
- Secure file transfer protocols
- Advanced SLURM job submission
- Software installation and environment management
- Data encryption and access controls
- Compliance with data protection regulations

## Prerequisites

- Basic knowledge of HPC systems
- Familiarity with Linux command line
- Understanding of data sensitivity requirements
- Access to sensitive data projects (or understanding of requirements)

## Target Audience

- Researchers working with sensitive data
- Students handling confidential research data
- Staff supporting sensitive data projects
- Anyone requiring secure computing environments
