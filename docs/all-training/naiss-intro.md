# NAISS Introduction training days

Aimed at new users, covering: Linux, connecting, file transfer, software use and batch systems.

## Course Details

**Dates**: 1 - 3 June 2026  
**Location**: Online  
**Duration**: 3 days  
**Level**: Beginner

## Summary

This comprehensive introduction course is designed for new NAISS users. You'll learn the fundamental skills needed to effectively use Sweden's national HPC infrastructure.

## Learning objectives

- Navigate Linux command line confidently
- Connect to and use HPC clusters
- Transfer files securely
- Use software modules and environments
- Submit and manage jobs with SLURM

## Topics Covered

### Day 1: Getting Started
- Introduction to HPC and NAISS
- Linux command line basics
- Connecting to clusters (SSH)
- File system navigation

### Day 2: Data and Software
- File transfer techniques
- Software modules and environments
- Installing and using software
- Basic scripting

### Day 3: Job Submission
- SLURM batch system overview
- Writing job scripts
- Submitting and monitoring jobs
- Troubleshooting common issues

## Prerequisites

None! This course is designed for complete beginners.

## Target Audience

- New NAISS users
- Researchers starting with HPC
- Students learning computational research
- Anyone new to high-performance computing
