# R Programming for HPC and Data Analysis

Harness the power of R for large-scale data analysis and statistical computing on HPC systems.

## Course Details

**Dates**: TBD  
**Location**: Online  
**Duration**: 3 days  
**Level**: Intermediate

## Summary

This course teaches how to scale R applications for HPC environments, covering parallel computing, memory management, and integration with HPC workflows. You'll learn to handle large datasets and computationally intensive statistical analyses efficiently.

## Learning objectives

- Scale R applications for HPC environments
- Implement parallel computing in R
- Optimize memory usage and performance
- Integrate R with HPC job schedulers
- Use R for big data analytics
- Apply statistical computing at scale

## Topics Covered

### Day 1: R on HPC Fundamentals
- R environment setup on HPC clusters
- Package management and environments
- Job submission with R scripts
- Memory management and optimization
- File I/O and data handling at scale

### Day 2: Parallel Computing in R
- Parallel processing with parallel package
- foreach and doParallel for loops
- Multiprocessing and multicore computing
- Distributed computing with R
- GPU acceleration with gpuR

### Day 3: Advanced R for Big Data
- Large dataset handling with data.table
- Statistical computing at scale
- Integration with HPC workflows
- Performance profiling and optimization
- Reproducible research practices

## Prerequisites

- Basic R programming experience
- Understanding of statistical concepts
- Familiarity with Linux command line
- Basic knowledge of HPC concepts

## Target Audience

- Bioinformaticians and computational biologists
- Data scientists and statisticians
- Researchers working with large datasets
- Anyone using R for computationally intensive work

## R Packages Covered

- **Parallel Computing**: parallel, doParallel, foreach, future
- **Big Data**: data.table, dplyr, dtplyr, arrow
- **Statistics**: glmnet, randomForest, xgboost, caret
- **Visualization**: ggplot2, plotly, shiny
- **HPC Integration**: batchtools, clustermq, rslurm

## Materials

- R scripts and examples for HPC
- Parallel computing exercises
- Performance optimization guides
- Integration examples with SLURM
