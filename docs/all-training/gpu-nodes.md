# Efficient use of the Grace Hopper GPU nodes on Dardel and Arrhenius

Efficient use of the Grace Hopper GPU nodes on Dardel and Arrhenius.

## Course Details

**Dates**: 30 - 31 March 2026  
**Location**: Online  
**Duration**: 2 days  
**Status**: Completed

## Summary

This advanced course focused on efficiently using the Grace Hopper GPU nodes available on Sweden's Dardel and Arrhenius supercomputers. Participants learned optimization techniques and best practices for GPU computing.

## Topics Covered

### Day 1: Grace Hopper Architecture
- Grace Hopper GPU specifications
- Memory hierarchy and bandwidth
- Programming model overview
- Performance characteristics

### Day 2: Optimization Techniques
- CUDA optimization strategies
- Memory access patterns
- Kernel launch optimization
- Multi-GPU programming
- Profiling and debugging
