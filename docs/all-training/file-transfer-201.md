# File Transfer 201

Transfer with terminal tools, including cluster to cluster.

## Course Details

**Date**: 11 May 2026  
**Location**: Online  
**Duration**: 1 day  
**Level**: Intermediate

## Summary

Advanced file transfer techniques for HPC environments, including cluster-to-cluster transfers and large data management.

## Learning objectives

- Master command-line file transfer tools
- Understand cluster-to-cluster data movement
- Learn to handle large datasets efficiently
- Implement secure transfer protocols
- Automate transfer workflows

## Topics Covered

- SCP and SFTP advanced usage
- Rsync for efficient synchronization
- Globus for large-scale transfers
- Cluster interconnect transfers
- Data integrity verification
- Automation with scripts

## Prerequisites

- Basic Linux command line experience
- Understanding of file systems
- Familiarity with SSH

## Target Audience

- Researchers moving data between systems
- HPC users managing large datasets
- System administrators
- Anyone working with distributed computing
