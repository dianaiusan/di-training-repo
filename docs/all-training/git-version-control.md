# Version Control and Reproducible Research with Git

## Summary

This course introduces Git as a tool for tracking changes to code and data, collaborating with others, and making research reproducible. Participants will learn how to set up a repository, manage branches, and integrate Git into their HPC workflows.

## Learning objectives

- Understand the principles of version control
- Create and manage Git repositories
- Use branches, commits, and merges effectively
- Collaborate using remote repositories (GitHub/GitLab)
- Apply Git to HPC scripts and analysis code for reproducibility
