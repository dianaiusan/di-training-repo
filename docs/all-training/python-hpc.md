# Introduction to Python and Using Python in an HPC environment

A brief, but comprehensive introduction to using Python in a Swedish academic High-Performance Computing (HPC) environment.

## Course Details

**Dates**: 20, 22, 23 and 24 April 2026  
**Location**: Online  
**Duration**: 4 half-days  
**Level**: Beginner to Intermediate

## Summary

This course provides a comprehensive introduction to using Python in Sweden's academic HPC environment. You'll learn both Python programming fundamentals and how to effectively use Python on HPC clusters.

## Learning objectives

- Master Python programming basics
- Understand HPC cluster environments
- Learn to submit Python jobs to SLURM
- Use Python scientific libraries effectively
- Optimize Python code for HPC systems

## Schedule

### Day 1: Python Fundamentals
- Python syntax and data structures
- Control flow and functions
- Working with files and data

### Day 2: Scientific Python
- NumPy for numerical computing
- Pandas for data manipulation
- Matplotlib for visualization

### Day 3: Python on HPC
- Connecting to HPC clusters
- Environment modules and Python
- SLURM job submission for Python

### Day 4: Advanced Topics
- Parallel Python with multiprocessing
- Performance optimization
- Best practices for HPC Python

## Prerequisites

- Basic understanding of programming concepts
- Familiarity with Linux command line (helpful but not required)

## Target Audience

- Researchers wanting to use Python for their work
- Students learning scientific computing
- Staff supporting research computing
