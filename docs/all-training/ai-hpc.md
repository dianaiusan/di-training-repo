# AI and HPC

This hybrid (on location/online) course is at the beginners/intermediate level and is mainly intended for students familiar with HPC and in need of an introduction to Deep Learning and how to run AI workloads on HPC clusters.

## Course Details

**Dates**: 3 - 5 June 2026  
**Location**: Stockholm on-site and online  
**Duration**: 3 days  
**Level**: Beginner to Intermediate

## Summary

This hybrid course introduces deep learning and AI workloads on HPC clusters. You'll learn how to effectively run machine learning applications on Sweden's national supercomputers.

## Learning objectives

- Understand AI/ML on HPC systems
- Learn GPU computing fundamentals
- Master deep learning frameworks on clusters
- Optimize AI workloads for performance
- Use HPC resources for machine learning

## Topics Covered

### Day 1: AI on HPC Fundamentals
- Introduction to AI/ML on supercomputers
- GPU architecture and CUDA basics
- Deep learning frameworks (TensorFlow, PyTorch)
- Containerized AI environments

### Day 2: Running AI Workloads
- SLURM job submission for AI
- GPU resource allocation
- Multi-node training
- Performance optimization

### Day 3: Advanced Topics
- Large-scale model training
- Distributed deep learning
- AI workflow automation
- Best practices and case studies

## Prerequisites

- Basic HPC experience (connecting, job submission)
- Familiarity with Python programming
- Basic understanding of machine learning concepts

## Target Audience

- Researchers using AI/ML methods
- Students with HPC experience
- Developers optimizing AI applications
- Anyone running computational AI workloads

## Format

- **Hybrid**: In-person in Stockholm + online participation
- **Hands-on**: Practical exercises on real HPC systems
- **Interactive**: Q&A sessions and collaborative work
