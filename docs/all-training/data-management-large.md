# Large Dataset Management for HPC

## Summary

Research datasets can quickly grow beyond what can be managed with simple file copies. This course covers strategies for handling large datasets on NAISS systems: efficient storage formats (HDF5, Zarr, Parquet), dataset versioning with DVC, metadata management, and sharing datasets with collaborators or public repositories. Participants will work with realistic scientific datasets throughout.

## Learning objectives

- Choose appropriate file formats for large structured and unstructured data
- Use DVC (Data Version Control) to track and share datasets alongside code
- Organise project directories for reproducibility and longevity
- Transfer and stage large datasets efficiently across NAISS storage tiers
- Publish datasets to data repositories (e.g. Zenodo, SciLifeLab Data Repository)

## Prerequisites

- Basic HPC familiarity (naiss-intro)
- Version control basics (git-version-control)
