# Array computing in C++ using Eigen

This course introduces students to high-performance array computing in C++ using the Eigen library, a versatile template library for linear algebra, matrices, and vectors.

## Course Details

**Date**: 5 March 2026  
**Location**: Online  
**Duration**: 1 day  
**Status**: Completed

## Summary

This course introduced high-performance array computing in C++ using the Eigen library. Participants learned to leverage Eigen's powerful template metaprogramming for efficient linear algebra operations.

## Topics Covered

- Eigen library fundamentals
- Matrix and vector operations
- Template metaprogramming concepts
- Performance optimization techniques
- Integration with existing C++ code
- Advanced Eigen features (sparse matrices, decompositions)
