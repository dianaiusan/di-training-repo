# Basic Singularity/Apptainer workshop

An introduction to the basic concepts of containerised software environment solution within the Singularity/Apptainer framework (run, build, etc.).

## Course Details

**Date**: 9 February 2026  
**Location**: Online  
**Duration**: 1 day  
**Status**: Completed

## Summary

This workshop introduced the fundamental concepts of containerized software environments using Singularity/Apptainer. Participants learned how to run, build, and manage containers for reproducible research computing.

## Topics Covered

- Container concepts and benefits
- Singularity/Apptainer basics
- Running existing containers
- Building custom containers
- Integration with HPC workflows
- Best practices for scientific containers
