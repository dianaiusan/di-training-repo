# Explore Section Generators

The explore section pages are generated by scripts under `scripts/generators/`.

Generated page groups:
- `docs/explore/scientific-domains/`
- `docs/explore/learning-paths/`
- `docs/explore/bundles/`
- `docs/explore/tags/`

Main commands:

```bash
.venv/bin/python scripts/generators/generate_scientific_domains.py
.venv/bin/python scripts/generators/generate_learning_paths.py
.venv/bin/python scripts/generators/generate_bundles.py
.venv/bin/python scripts/generators/generate_tags.py
```

Run all current generators + build:

```bash
.venv/bin/python scripts/generate_all.py
```

Color/style consistency:
- Card-grid hover and accent colors are shared via CSS tokens in `docs/assets/stylesheets/extra.css`.
- Scientific domains style is treated as the visual reference for overview cards.
